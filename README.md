# 📄 Local PDF Chatbot

An AI-powered chatbot that allows users to ask questions about PDF documents.

## 🚀 Features

- Upload PDF documents
- Ask questions about the document
- Local AI processing using Ollama
- Private and free AI chatbot
- Retrieval Augmented Generation (RAG)

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Ollama
- Llama 3.1
- Nomic Embeddings
